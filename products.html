Description

The SinoTrack ST-901A GPS Vehicle Tracker is a reliable real-time vehicle tracking device suitable for cars, motorcycles, vans, trucks and fleet management. Monitor your vehicle anytime using the Android App, iOS App, Web Platform or SMS commands. Track accurate live location, ignition status, travel history, speed, mileage and many other vehicle parameters from anywhere. It also supports remote engine cut-off (optional), geo-fence alerts, overspeed notifications and intelligent alarms for better vehicle security.

Main Features

  • Real-Time Live Location Tracking
  • Android & iOS Application Support
  • Web Tracking Platform
  • GPS + GSM Positioning
  • Remote Engine Cut-Off (Optional)
  • ACC Ignition Status Monitoring
  • Travel History Playback
  • Mileage Report
  • Speed Monitoring
  • Overspeed Alarm
  • Geo-Fence
